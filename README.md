# C Advanced Recursion

This repository contains advanced C programs that demonstrate recursion concepts and problem-solving techniques. These examples help in understanding recursive function calls, traversal orders, maze path exploration, and the Tower of Hanoi algorithm.

## Programs Included

1. **maze2.c**
   - Solves maze path problems using recursion.

2. **preinpost.c**
   - Demonstrates pre-order, in-order, and post-order execution in recursion.

3. **zigzag.c**
   - Prints a zigzag recursion pattern to understand recursive call flow.

4. **tower.c**
   - Solves the Tower of Hanoi problem using recursion.

5. **arraybasics.c**
   - Demonstrates basic array operations and recursive array concepts.

## Concepts Covered

- Advanced Recursion
- Recursive Call Flow
- Pre, In & Post Order
- Maze Path Problems
- Tower of Hanoi
- Zigzag Recursion
- Array Basics
